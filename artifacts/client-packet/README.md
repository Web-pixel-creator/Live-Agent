# Visa Client Packet

This folder is the single handoff point for the current commercial wedge:

- visa agencies
- relocation agencies
- cross-border intake teams

Use this packet when you need one place that explains:

1. what the product is,
2. how to pitch it,
3. how to present it,
4. what to send to a client,
5. how to run the live demo.

## Product Name

`AI Action Desk`

## Best Short Description

`AI Action Desk is an operator-safe agent for visa and relocation teams that qualifies leads, books consultations, follows up on documents, and prepares CRM or human handoff actions from one workspace.`

## Public Demo

- [live-agent-frontend-production.up.railway.app](https://live-agent-frontend-production.up.railway.app)

## Best Demo Path

1. `Run Visa Intake Demo`
2. `Review Visa Draft Result`
3. `Run Missing Docs Follow-up`
4. `Review Follow-up Result`
5. `Run Consultation Reminder`
6. `Review Reminder Result`
7. `Run CRM Update Demo`
8. `Review CRM Handoff Result`
9. `Run Case Escalation`
10. `Review Human Handoff Result`

## Recommended Send Order

For first outreach:

1. outreach email,
2. one-pager,
3. public demo link.

For a live meeting:

1. founder pitch,
2. one-pager,
3. deck copy or deck outline,
4. live demo,
5. sales call script as talk track.

For investor or partner conversations:

1. founder pitch,
2. deck copy,
3. one-pager,
4. public demo link,
5. operator walkthrough if they want a guided live run.

## What To Emphasize

Always emphasize:

1. `visa and relocation teams`
2. `operator-safe`
3. `lead qualification`
4. `booking`
5. `document follow-up`
6. `CRM handoff`
7. `human escalation`

Avoid leading with:

1. `generic AI platform`
2. `storytelling`
3. `challenge categories`
4. `multimodal lab`

## Quick Copy Block

Use this if you need one paragraph fast:

`AI Action Desk helps visa and relocation teams move a client case from first contact to human handoff. The agent qualifies leads, books consultations, follows up on missing documents, prepares CRM updates, and escalates difficult cases to the right human owner, while keeping approval boundaries and verified summaries visible to the operator.`
